How to Use FIFO Dispatch
✓ What to Do:
  1. Look at "Required Batch" column - this shows the EXACT batch code you must pick
  2. Go to warehouse shelf and find products with THAT EXACT batch code
  3. Scan the batch barcode on the product you picked
  4. If correct, row turns green ✓
  5. Repeat for each product in the order
✗ If You See "WRONG BATCH":

This means you picked a product with the WRONG batch code!

  • DO NOT scan different products - scan the SAME product but with the correct batch code
  • PUT BACK the product you just picked
  • FIND the product with the batch code shown in "Required Batch" column
  • SCAN the correct batch - older batches must be used first!
Important: You MUST use the oldest batch first (FIFO = First In, First Out). If you picked a newer batch, put it back and get the older one shown in "Required Batch".
